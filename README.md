# Vue Cursor Fx

[DEMO](https://luxdamore.github.io/vue-cursor-fx)

## Development Setup

``` bash
# install deps
npm install

# build dist files
npm run build

# serve examples at localhost:8080
npm run dev

# lint & run all tests
npm test

# serve docs at localhost:8080
npm run docs
```
